Python program for creating followpoints

# Usage
+ Download and run executable file from the releases
+ Import skin osk file or use skins from osu folder using list menu (optional)
+ Make your followpoints
+ Export your followpoints to specified directory or update followpoints in current preveiw skin using Export button

# Settings guide

+ Main Color", "The primary color of the followpoint and Glow effect.
+ Shadow Color", "The color used for the Shadow.
+ Thickness", "The height of the followpoint in pixels.
+ Length", "The width of the followpoint max 128px.
+ Glow", "Adds a soft glow around the followpoint.
+ "Shadow", "Adds a shadow behind the followpoint.
+ Shadow opacity", "Controls the opacity of the shadow.
+ Edge Fade", "Fades the left and right side of the followpoint.
+ Base Opacity", "The overall transparency level of the followpoint.
+ Min Transparency", "Controls the fade-in gradient at the start of the followpoint path.
+ Max Transparency", "Controls the fade-out gradient at the end of the followpoint path.
+ Roundness", "Adjusts the corners of the followpoint.
+ Transparent Files", "The number of blank files exported from the start and end of the followpoints images from 0 to 31 and 60 to 32.
+ Circle Count/Dist", "Adjusts the preview (IN PROGRESS).
+ Export", "Export followpoints to an specified directory or choosed skin folder.
