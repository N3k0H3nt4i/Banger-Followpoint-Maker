Python program for creating followpoints

# Usage
+ Download and run executable file from the releases
+ Import skin osk file or use skins from osu folder using list menu (optional)
+ Make your followpoints
+ Export your followpoints to specified directory or update followpoints in current preveiw skin using Export button
